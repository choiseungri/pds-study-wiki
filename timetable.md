# PDS 배치표 원본 전사

사용자가 제공한 시간표 이미지 기준 전사본입니다. 괄호 속 이름을 담당 교수님으로 사용하고, `/` 뒤 이름은 발표자 또는 보조 표기로 둡니다. 판독이 애매한 발표자 이름은 사이트 태그에 쓰지 않습니다.

## 사용 규칙

- 홈 화면 태그는 `professor`만 사용합니다.
- 2시간짜리 강의는 시작 교시의 HTML 파일명만 만들고, `site.config.json`의 `pageSpans`에 `2`를 지정합니다.
- 배포 파일명은 `MMDD_교시_english-slug.html` 형식입니다.
- 현재 HTML이 없는 항목은 파일명을 비워두고, 자료가 생기면 이 표를 기준으로 추가합니다.
